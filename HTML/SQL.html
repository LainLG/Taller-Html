SQL

SQL (por sus siglas en inglés Structured Query Language; en español lenguaje de consulta estructurada)
es un lenguaje de dominio específico, diseñado para administrar, y recuperar información de sistemas
de gestión de bases de datos relacionales.Una de sus principales características es el manejo del álgebra
y el cálculo relacional para efectuar consultas con el fin de recuperar, de forma sencilla, información de
bases de datos, así como realizar cambios en ellas
